## Требования

- Python 3.11
- Poetry (для управления зависимостями)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/KissedByF1re/people-detection.git
   cd people-detection

2. Установите зависимости с помощью Poetry:

```
poetry install
```


### Запуск проверок кода 

```
poetry run ruff check .
```

Автоматически исправить ошибки
```
poetry run ruff fix .
```
