# Проект: Система детекции людей на видео

## Цель проекта

Создать прототип системы для детекции людей на видео. Система должна точно и быстро распознавать людей в различных условиях освещения и погоды.

## Ожидаемые результаты

1. **Демонстрационный проект**, позволяющий оценить работу алгоритма.
2. **Отчет** (в формате `README.md`), содержащий полную информацию о ходе работы.
3. **Репозиторий** с кодом проекта, включая код обучения модели (или моделей).

## Критерии оценки

### 1. Выбор технологий и инструментов

- Какие фреймворки и библиотеки машинного обучения были использованы и почему.
- Какие методы и модели применялись и насколько они соответствуют поставленной задаче.

### 2. Обработка данных

- Описание процесса сбора и подготовки датасета.
- Методы аугментации данных, использованные для повышения робастности модели.

### 3. Обучение модели

- Формирование обучающего, валидационного и тестового наборов данных.
- Используемые метрики для оценки качества модели: accuracy, precision, recall, F1-score и т.д.

### 4. Тестирование и валидация

- Подходы к тестированию модели и критерии успешности.
- Представление информации о проведённых экспериментах.

### 5. Интеграция и деплой

- Качество и структура кода.
- Интеграция решения в прототип и взаимодействие с пользовательским интерфейсом.

### 6. Продуктовые детали

- Сценарии использования и условия, для которых разрабатывается система.
- Варианты интеграции с существующими платформами.
- Сценарии инференса и взаимодействия пользователя с системой.
- Оптимизация модели для работы в реальном времени.

## Датасеты для детекции людей

### 1. [Caltech Pedestrian](https://www.kaggle.com/code/harits/people-detection-and-tracking-yolov8-bytetrack/notebook)
- 10 часов уличного видео.
- 250 000 кадров, 350 000 «обрамляющих прямоугольников» для пешеходов.
- На Kaggle доступен ноутбук с обучением и, возможно, сам датасет.

### 2. [CityPersons](https://www.kaggle.com/code/shomerthesec/yolo-v8-for-pedestrians-detection-citypersons)
- Основан на Cityscapes.
- Уличные сцены с человекоподобными объектами.
- 2975 кадров для обучения, 500 — для валидации, 1575 — для тестирования.
- В среднем — 7 пешеходов на кадр.
- Доступен ноутбук с обучением.

### 3. [NightOwls](https://www.nightowls-dataset.org/)
- Аннотации пешеходов на ночных городских кадрах.
- Особенно полезен для оценки работы моделей в условиях низкой освещённости.
- В оригинальном описании указано, что большинство современных моделей плохо работают ночью.

### 4. [INRIA Person](https://paperswithcode.com/dataset/inria-person#:~:text=The%20INRIA%20Person%20dataset%20is,training%20and%20288%20for%20testing)
- Классический датасет пешеходов от Dalal & Triggs (2005).
- Содержит изображения стоящих людей в различных позах и на разных фонах.
- Также включает «негативные» изображения (без людей) для обучения.

### 5. [Pascal VOC](https://huggingface.co/datasets/merve/pascal-voc)
- Бенчмарк для общей детекции объектов с 20 классами, включая «person».
- Изображения из Flickr и других источников, охватывают разнообразные реальные сцены.

### 6. [MS COCO (Person subset)](https://paperswithcode.com/datasets#:~:text=MS%20COCO%20,11%2C603%20PAPERS%20•%2096%20BENCHMARKS)
- Категория «person» включает тысячи объектов на множестве изображений (спорт, улицы и т.п.).
- Представлены разнообразные условия: улица/помещение, различное освещение.

### 7. [MOT17 (Pedestrian Detection)](https://motchallenge.net/data/MOT17Det/#:~:text=Image%20MOT17,0%20People%20walking%20around%20a)
- Содержит 14 коротких видеопоследовательностей (7 для обучения, 7 для теста).
- Сцены с большим скоплением людей, используется для задач трекинга и детекции.
